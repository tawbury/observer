비개발자가 AI 도구들(Claude Code, Windsurf, GitHub Copilot, ChatGPT, Gemini, Claude 등)과 협업하여 주식 트레이딩 시스템을 개발하고 운영하기 위한 문서화 가이드입니다.

## 🎯 프로젝트 개요

### 시스템 구조

```
주식 트레이딩 시스템 (Stock Trading System)
│
├─ 프로젝트 1: 데이터 수집 & 아카이빙 시스템
│  └─ 증권사 API → 데이터 추출 → 원본 아카이브 → DB 저장
│
└─ 프로젝트 2: 자동매매 프로그램
   └─ DB 데이터 분석 → 매매 신호 생성 → 증권사 API 주문 실행
```

## 1️⃣ 프로젝트 1: 데이터 수집 & 아카이빙 시스템

### 서비스 유형 분류

```
주 유형: 데이터 파이프라인 (Data Pipeline/ETL)
부 유형: 백엔드 서비스 (API Integration)
```

**왜 이 유형인가?**
- 외부 API에서 데이터를 주기적으로 수집 (Extract)
- 원본 데이터를 아카이브 로그로 보관 (Transform)
- 원하는 형태로 DB에 적재 (Load)
- 전형적인 ETL 파이프라인
