# Cross-Agent Validator

## Purpose
- Inter-agent collaboration validation
- Cross-functional work quality assessment
- Multi-agent project integration verification

## Validation Criteria
### Collaboration Quality
- Inter-agent communication effectiveness
- Shared understanding verification
- Dependency management
- Conflict resolution approaches

### Integration Excellence
- Cross-agent workflow consistency
- Shared standards compliance
- Interface compatibility
- End-to-end quality assurance

### Synergy Assessment
- Combined value creation
- Resource optimization across agents
- Innovation through collaboration
- Scalability of integrated solutions

## Validation Process
1. Cross-agent project structure review
2. Integration points verification
3. Communication channel assessment
4. Quality standards consistency check
5. Synergy effectiveness measurement

## Agent-Specific Checks
- Developer ↔ PM: Technical feasibility ↔ Business requirements
- Developer ↔ Finance: Technical costs ↔ Financial planning
- Contents-Creator ↔ PM: Creative requirements ↔ Product strategy
- All agents ↔ HR: Role requirements ↔ Performance evaluation

## Integration
- Connect to all agent skill matrices
- Link to workflow systems
- Integrate with existing validators
- Align with project management frameworks
