# 프로젝트 문서

- **guides/** — 운영·배포·롤백 가이드
- **dev/** — 아키텍처, PRD, 개발 가이드
- **deployment/** — 배포 관련 문서
- **tasks/** — 태스크·페이즈 정리
- **refactoring/** — 리팩터링 로드맵·태스크
- **reports/** — 분석·감사·운영 리포트

**archive/** — 과거 완료·히스토리 문서 보관 (`archive/completion_reports/` 등).  
최신 가이드는 **guides**, **dev**, **deployment**를 참고하세요.

모니터링 설정(그라파나, 프로메테우스 등)은 `infra/monitoring/`을 참고하세요.
